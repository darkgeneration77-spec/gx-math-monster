# GX Sejarah Tingkatan 4 — Question Bank Standard

## 核心规则

1. 只收录 Tingkatan 4 对应 Bab 的内容，不混入 Tingkatan 5。
2. 每题必须有唯一 ID，例如 `W1-1.1-001`。
3. 每题必须标明：
   - route / subtopic
   - difficulty
   - mode
   - paper
   - marks
   - Bahasa Melayu keywords
   - Chinese explanation
   - hint
   - correction reminder
4. `Nyatakan`：每1分对应1个不同事实。
5. `Jelaskan / Huraikan`：必须包含 Fakta + Huraian。
6. KBAT：答案必须合理、不重复，并直接回应 kata tugas。
7. Esei Builder：不能只提供零散关键词，必须能组成完整考试答案。
8. 干扰项必须来自常见错误，但不可包含明显荒谬或超纲内容过多。
9. 选项顺序在游戏中应随机，不可让学生记答案位置。
10. 错题报告必须显示：华语解释、BM关键词、订正提醒。

## 难度分层

- `easy`：识别课本事实、人物、定义、名称。
- `standard`：配对、比较、解释、结构题。
- `exam`：KBAT、Esei、综合证据链、Boss。

## 评分规则

### Objective
- 1题1分。

### Struktur
- 题目几分，就必须有相同数量的有效得分点。
- 选择超过要求数量时，不直接判满分。

### Huraian Builder
- Fakta 正确：1分。
- 对应 Huraian 正确：1分。
- 重复含义不得重复给分。

### Esei Constructor
- 依卡片标签显示 Fakta / Huraian / Kesan / Inferens / Penegasan。
- 最终输出必须自动组合成正式段落。
